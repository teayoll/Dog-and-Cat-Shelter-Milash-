# Use Case: Система приюта «Милаш»

# Публичная часть (гость)

## UC-01: Просмотр главной страницы (о приюте)

### Краткое описание
Гость может просматривать информацию о приюте.

### Акторы
- Гость

### Предусловия
- Пользователь открыл сайт

### Основной сценарий
1. Пользователь открывает главную страницу
2. Система отображает информацию о приюте
3. Система отображает блоки новостей и питомцев (если предусмотрено)

### Постусловия
- Данные не изменяются
